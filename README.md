# QtNFCReaderGUI

## About
This project is a simple GUI for using Mifare Classic USB cards.

## Installation

### Requirements
To compile this project, you need :
* Visual Studio 2019 or later
* Qt 5.12.2

### Project setup
1. Download this repository
2. Download and install Qt 5.15.2
